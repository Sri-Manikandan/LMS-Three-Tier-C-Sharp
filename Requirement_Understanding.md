# Table Creation

## Member
- Add, view, search member by phone, or email
- Update membership status
- Deactivate a member
- Membership types: Basic, Premium and Student

## Book Management
- Add single, multiple copies
- view,search by title, author, category
- mark book as damaged

## Borrowing Rules
Cannot borrow a book
- Membership is inactive
- Reached the borrowing limit
- Book copy is not available
- Unpaid fees above 500
- Borrowed the same book and have not returned it

## Return Rules

- Calculate late return fine
- Fine amount = 10rs per day
- Returned copy should become available again
- Borrowing status should be changed to return
- Return date should be saved

## Fine Management

- View pending fines of a member
- Pay fine
- View fine history
- Prevent borrowing if unpaid fine is above 500
